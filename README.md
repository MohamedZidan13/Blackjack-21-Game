# Blackjack Game

A Python implementation of the classic Blackjack (21) card game.

## Features

* Realistic Blackjack gameplay
* Automated dealer behavior
* Random card dealing system
* Dynamic Ace value handling (1 or 11)
* Score calculation and winner determination
* Interactive command-line interface
* Multiple game rounds

## Technologies Used

* Python
* Random Module

## Skills Demonstrated

* Game logic implementation
* Conditional programming
* Randomization techniques
* User interaction design
